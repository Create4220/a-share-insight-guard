# 项目验收清单

## 基础运行

- [ ] 前端可通过 `npm install && npm run dev` 启动
- [ ] 后端可通过 `pip install -r requirements.txt && uvicorn app.main:app --reload` 启动
- [ ] 后端 Swagger 文档可访问
- [ ] 前端可构建成功
- [ ] 后端测试可运行
- [ ] 不需要任何付费 API Key

## 核心功能

- [ ] 可以输入一段金融分析文本
- [ ] 可以选择低、中、高风险演示文本
- [ ] 可检测直接荐股表达
- [ ] 可检测收益承诺
- [ ] 可检测确定性涨跌预测
- [ ] 可检测无来源事实类风险
- [ ] 可检测风险揭示缺失
- [ ] 可生成透明的评分和评分说明
- [ ] 可展示主张分类
- [ ] 可展示演示证据来源
- [ ] 可显示合规化改写建议
- [ ] 可提交人工审核
- [ ] 可在审核队列查看任务
- [ ] 可进行通过、退回、拦截、要求补证据操作
- [ ] 每个审核操作都有审计日志
- [ ] 可在规则中心启用/禁用规则
- [ ] 禁用规则会影响后续审校

## 合规要求

- [ ] 全局有免责声明
- [ ] 所有数据清楚标注为演示数据
- [ ] 不出现真实荐股或收益承诺
- [ ] 不提供交易或模拟交易功能
- [ ] 不展示实时行情、K 线或交易信号
- [ ] 高风险测试文本明确标注为检测样例
- [ ] README 有免责声明

- [ ] 有基础测试
- [ ] GitHub Actions 可执行构建/测试
